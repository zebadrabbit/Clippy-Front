# Analytics System - User Guide

## What is the Analytics Dashboard?

The Analytics Dashboard helps you understand how your community engages with your clips. It tracks views, identifies top creators and games, finds viral content, and shows engagement trends over time.

**Access**: Click "Analytics" in the main navigation bar (graph icon)

---

## Dashboard Overview

### Top Statistics Cards

When you open the dashboard, you'll see four key metrics at the top:

1. **Total Clips** - How many clips have been collected across all your projects
2. **Total Views** - Combined Twitch views for all your clips
3. **Unique Creators** - Number of different people who created clips
4. **Games Played** - Variety of games being clipped

These numbers update based on the time period you select.

### Time Period Selector

Filter all analytics by clicking the period buttons:
- **Day** - Last 24 hours
- **Week** - Last 7 days
- **Month** - Last 30 days
- **All Time** - Everything since you started using ClippyFront

- Updates when you re-fetch clips for a project

**Discord Integration**
- Shares counted when clips posted to configured channels

### What Counts as a "Clip"?

### How Often Does It Update?

- **Real-time**: When you add clips to projects

- **Dashboard**: Instantly when you change period filters

## Tips for Better Insights

## Common Questions

### Why are my view counts low?

- Clips need time to accumulate views
- Re-fetch clips after a few days for updated counts
- Share clips on social media to boost visibility

### Why don't I see any creators?

- You need to have fetched clips via Twitch integration
- Ensure clips have creator metadata (some old clips don't)
- Check that Twitch API connection is working

### Why is my game data incomplete?

- Twitch API sometimes doesn't include game for very old clips
- Manually added clips might not have game info
- Re-fetch from Twitch to enrich metadata

### Can I export this data?

Currently, analytics are view-only in the dashboard. Export features are planned for future releases.

### How far back does data go?

Analytics start from when clips are first added to your projects. Historical data before using ClippyFront is not available.
