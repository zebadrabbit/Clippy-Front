# Quick Reference: Using the Help System

## Accessing Help

### Global Help Icon
Click the **question mark icon** (?) in the top-right navigation bar to access the Help Center.

### Contextual Help
Look for **question mark icons** next to confusing fields or features:
- **Hover** over small icons for quick tooltips
- **Click** larger help buttons for quick-start guides with links to full documentation

### Direct URLs
- Help Center: `/help`
- Specific topic: `/help/topic-name`
- Search: `/help/search?q=your-query`

## Using Search

1. Click the help icon in navigation
2. Enter keywords in the search box
3. Press Enter or click Search
4. Click any result to view the full topic

Search looks through all help content, including:
- Topic titles
- Headers and sections
- Body text and examples

## Tips

- Use contextual help for quick answers
- Visit full help topics for detailed guides
- Search when you can't find what you need
- Help content updates with new features

## Need More Help?

If you can't find what you need in the help system:
1. Check the FAQ first
2. Try searching with different keywords
3. Contact your administrator for additional support
