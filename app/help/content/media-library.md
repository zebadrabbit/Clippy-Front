# Managing Your Media Library

<!-- category: media -->

Your media library is your personal collection of video files, intros, outros, and transitions.

## Accessing Your Library

Click **Media** in the main navigation to view your library.

## Media Types

Files in your library are categorized:

### Intro
Opening sequences for your compilations
- Recommended: 3-5 seconds
- Should grab attention quickly
- Examples: animated logo, channel intro

### Clip
Main content for compilations
- Downloaded from Twitch/Discord
- Uploaded manually
- Organized by tags and dates

### Transition
Short animations between clips
- Recommended: 1-3 seconds
- Keep them subtle and fast
- Examples: fade, swipe, zoom

### Outro
Closing sequences
- Recommended: 3-5 seconds
- Great for call-to-actions
- Examples: subscribe animation, social links

### Compilation
Final rendered videos
- Output from completed projects
- Available for download
- Can be reused in other projects

## Uploading Media

### Upload Individual Files

1. Click **Upload Media** button
2. Select media type (intro, clip, transition, outro)
3. Choose file(s) from your computer
4. Add optional tags and description
5. Click **Upload**

### Supported Formats

- **Video**: MP4, MOV, AVI, MKV, WebM
- **Maximum size**: Depends on your tier
- **Recommended**: MP4 with H.264 codec

### Bulk Upload

Upload multiple files at once:
1. Select multiple files in the upload dialog
2. All files receive the same type and tags
3. Edit individual files after upload if needed

## Downloading from Sources

### From Twitch

Clips are automatically downloaded when you:
1. Use the Project Wizard with Twitch route
2. Fetch clips in the "Get Clips" step
3. System checks for duplicates and reuses existing files

### From Discord

URLs are extracted and downloaded when you:
1. Use Discord route in Project Wizard
2. Provide date range for message scanning
3. System downloads each unique URL once

### Manual Downloads

You can also download clips directly:
1. Go to Media Library
2. Click **Download from URL**
3. Paste video URL
4. Select media type
5. Click **Download**

## Organizing Media

### Tags

Tags help you find files quickly:

**Adding Tags:**
1. Upload with tags, or
2. Edit file later to add tags
3. Use comma-separated values
4. Examples: funny, epic, fails, wins

**Searching by Tags:**
- Use the search box in Media Library
- Type tag name to filter
- Combine multiple tags for precise search

### Descriptions

Add notes to files:
- Remind yourself what happens in the clip
- Note who created it or when
- Add any other useful information

### Filtering

Filter your library by:
- **Media Type**: Show only intros, clips, etc.
- **Tags**: Find files with specific tags
- **Date**: Sort by upload date
- **Duration**: Filter by length

## File Details

Click any file to view:

- **Thumbnail**: Preview image from video
- **Duration**: Length of the clip
- **File Size**: Storage used
- **Upload Date**: When added to library
- **Tags**: Associated tags
- **Description**: Your notes
- **Metadata**: Twitch/Discord info if available

## Editing Media

### Update File Info

1. Click on a file
2. Click **Edit** button
3. Change name, type, tags, or description
4. Click **Save**

### Regenerate Thumbnail

If thumbnail looks wrong:
1. Click on the file
2. Click **Regenerate Thumbnail**
3. New preview image is created

## Deleting Media

### Delete Individual Files

1. Click on a file
2. Click **Delete** button
3. Confirm deletion

**Warning**: This permanently removes the file and frees up storage.

### Used Files

Files used in projects:
- Are safe to delete after project completes
- Compilation still exists separately
- Deleting doesn't affect existing compilations

### Bulk Delete

Delete multiple files:
1. Select files using checkboxes
2. Click **Delete Selected**
3. Confirm bulk deletion

## Storage Limits

Your tier determines storage capacity:

| Tier | Storage Limit |
|------|---------------|
| Free | 5 GB |
| Basic | 25 GB |
| Pro | 100 GB |
| Enterprise | 500 GB+ |

**Check Usage:**
- View current usage at top of Media Library
- Progress bar shows percentage used
- Upgrade tier if you need more space

**Free Up Space:**
- Delete old compilations
- Remove unused clips
- Keep only your best content

## Best Practices

### Organize as You Go

- Tag files immediately after upload
- Use consistent naming conventions
- Add descriptions while details are fresh

### Reuse Content

- Build a library of quality intros/outros
- Keep good transitions for reuse
- Don't delete versatile clips

### Quality Over Quantity

- Don't hoard every clip
- Keep only your best content
- Regularly clean up old files

### Backup Important Files

- Download your best compilations
- Keep local copies of custom intros/outros
- Don't rely solely on cloud storage

## Troubleshooting

### File Won't Upload

- Check file format is supported
- Verify file isn't corrupted
- Ensure you have storage space available
- Try a smaller file or different format

### Thumbnail Not Showing

- Wait a moment for generation
- Click **Regenerate Thumbnail**
- Check video file isn't damaged

### Can't Find a File

- Use search with tags
- Check media type filters
- Sort by date if recently uploaded

### Preview Won't Play

- Refresh the page
- Try different browser
- Check network connection
- File may need to be re-uploaded
