# Getting Started with Clippy

<!-- category: getting-started -->

Welcome to Clippy! This guide will help you get started with creating your first video compilation.

## What is Clippy?

Clippy is a powerful video compilation platform that helps you:

- **Collect clips** from Twitch and Discord
- **Organize media** in your personal library
- **Create compilations** with intros, outros, and transitions
- **Automate workflows** with scheduled tasks
- **Collaborate** with teams

## Quick Start

### 1. Set Up Your Profile

Before creating your first project:

1. Go to **Profile** (click your user menu in the top right)
2. Add your **Twitch username** if you want to fetch clips from Twitch
3. Configure your **Discord integration** if you use Discord
4. Set your preferred **timezone** and **date format**

### 2. Build Your Media Library

Upload or download media files to use in your compilations:

- **Upload files** directly from your computer
- **Download clips** from Twitch or Discord
- Organize files with **tags** for easy searching
- Add **descriptions** to remember what each clip contains

See [Managing Your Media Library](media-library) for more details.

### 3. Create Your First Project

Ready to make a compilation? Use the **Project Wizard**:

1. Click **Projects** â†’ **New Project**
2. Choose your source (Twitch or Discord)
3. Enter project details (name, description, tags)
4. Select platform preset (YouTube, TikTok, Instagram, etc.)
5. Configure audio normalization if needed

See [Creating Projects](creating-projects) for a detailed walkthrough.

### 4. Fetch and Arrange Clips

The wizard guides you through:

1. **Get Clips**: Fetch clips from your configured sources
2. **Arrange**: Choose intro/outro, add clips, set transitions
3. **Timeline**: Drag to reorder, adjust durations
4. **Compile**: Start the video rendering process

## Key Concepts

### Projects

A project is a container for your compilation. It includes:

- Project metadata (name, description, tags, status)
- Platform settings (resolution, aspect ratio, framerate)
- Audio normalization preferences
- List of clips and their order

### Media Types

Clippy organizes media into categories:

- **Intro**: Opening sequence for your compilation
- **Clip**: Main content clips
- **Transition**: Short animations between clips
- **Outro**: Closing sequence
- **Compilation**: Final rendered output

### Tiers & Quotas

Your account tier determines:

- Maximum storage space
- Maximum compilation length
- Number of concurrent projects
- Output resolution limits
- Access to advanced features

See [Understanding Tiers](tiers-quotas) for more information.
