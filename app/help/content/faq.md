# Frequently Asked Questions

<!-- category: getting-started -->

Quick answers to common questions about using Clippy.

## Getting Started

### Do I need to connect Twitch or Discord?

Not required! You can:
- Upload videos manually
- Create compilations from uploaded files
- Integrations are optional features

### What video formats are supported?

Most common formats:
- MP4 (recommended)
- MOV, AVI, MKV, WebM
- H.264 codec works best

### How do I get more storage?

Contact your administrator to upgrade your tier:
- Free: 5 GB
- Basic: 25 GB
- Pro: 100 GB
- Enterprise: 500 GB+

## Projects & Compilations

### How long does compilation take?

Depends on:
- Total video length
- Number of clips
- Output resolution
- Server load

Typical: 1-2x real-time (10 min video = 10-20 min render)

### Can I edit after compilation starts?

No - you must:
1. Cancel current compilation
2. Edit project
3. Start new compilation

### What if compilation fails?

Check:
- Individual clips play correctly
- Clips aren't corrupted
- You have sufficient quota
- Try removing problematic clips

### Can I download the source clips?

Yes:
- Go to Media Library
- Click on any clip
- Download button available
- Original file is preserved

## Media Library

### Why are my clips organized by type?

Helps you find files quickly:
- **Intro**: Opening sequences
- **Clip**: Main content
- **Transition**: Between-clip animations
- **Outro**: Closing sequences
- **Compilation**: Final outputs

### Can I change a clip's type?

Yes:
1. Click on the file
2. Click **Edit**
3. Change **Media Type**
4. Save changes

### Do deleted files affect old compilations?

No:
- Compilations are independent files
- Deleting source clips is safe
- Final videos remain intact

### How do I organize thousands of clips?

Use tags:
- Tag files on upload
- Search by tag later
- Use consistent tag names
- Combine tags for filtering

## Integrations

### My Twitch username isn't working

Check:
- Spelling is exact
- No @ symbol needed
- Just the username
- Try lowercase

### Discord bot can't access my channel

Verify:
- Bot is in your server
- Channel ID is correct
- Bot has read permissions
- Channel isn't private

### Why aren't all my clips showing?

Limitations:
- Twitch API limits recent clips only
- Discord scans date range specified
- Some platforms block downloads
- Private content isn't accessible

### Can I download from YouTube?

Through Discord integration:
- Share YouTube links in Discord
- Scan channel messages
- System downloads from URLs
- Subject to YouTube terms

## Tiers & Limits

### What counts toward storage?

Counts:
- Uploaded files
- Downloaded clips
- Compilations

Doesn't count:
- Thumbnails
- Temporary files
- Project metadata

### I hit my project limit

Solutions:
- Complete existing projects
- Delete old drafts
- Upgrade your tier
- Archive finished work

### Can I export 4K video?

Tier requirements:
- **Enterprise tier** for 4K
- Pro tier: 1080p 60fps
- Basic tier: 1080p 30fps
- Free tier: 720p

### How do I check my limits?

Multiple locations:
- Dashboard overview
- Media Library header
- Profile page
- Each shows current usage

## Teams & Collaboration

### Can multiple people work on a project?

Pro tier and above:
- Create teams
- Share projects
- Collaborate on timelines
- Role-based permissions

### How do I share a compilation?

After rendering:
1. Download to computer
2. Upload to YouTube/platform
3. Or share download link (temporary)

### Can I let others use my intros?

With teams feature:
- Share media within team
- Set permissions
- Collaborative libraries
- Team-wide resources

## Technical Issues

### Video won't upload

Try:
- Check file format
- Verify file isn't corrupted
- Ensure storage space available
- Try smaller file first

### Compilation is low quality

Check:
- Your tier's resolution limit
- Platform preset selected
- Source clip quality
- Audio normalization settings

### Page won't load

Basic troubleshooting:
- Refresh page
- Clear browser cache
- Try different browser
- Check internet connection

### Lost my work

Auto-save features:
- Projects save automatically
- Drafts are preserved
- Check Projects page
- Contact admin if missing

## Account & Settings

### How do I change my password?

1. Go to **Profile**
2. Click **Account Settings**
3. Enter new password
4. Confirm changes

### Can I change my username?

Contact administrator:
- Username changes require admin
- Doesn't affect your content
- Projects remain yours

### What timezone should I use?

Set in Profile:
- Affects date displays
- Timestamp formatting
- Schedule automation timing
- Choose your local timezone

### How do I delete my account?

Contact administrator:
- Account deletion is admin-only
- All your content is removed
- Process is permanent
- Export important files first
